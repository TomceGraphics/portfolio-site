## THE HYPOTHESIS

Can we build a lightweight, native downloader that avoids the bloat of electron while maintaining a gorgeous UI?
**Yes we can!**
this project stared out of personal need for a music downloader that was not filled with ads and malware.


## TECHNICAL ANALYSIS

Built using native windows frameworks to ensure minimal RAM usage.

| Feature | Description |
| --- | --- |
| **Architecture** | Native Windows Application |
| **Performance** | Optimized for minimal RAM usage |
| **UI** | Modern, premium interface |
| **Downloads** | Multithreaded download engine |
| **Updates** | Automatic background updates |
| **Security** | No malware or adware |

## FINAL RESULT

"Success. The application is virtually unnoticeable in the background and downloading is multithreaded."
